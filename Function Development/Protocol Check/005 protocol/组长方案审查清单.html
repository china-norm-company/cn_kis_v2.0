组长方案审查清单

客户方案设计 → 研究员撰写具体方案 → 组长审查 → 通过后可最终交付

化妆品功效测试方案

流程：客户提供方案设计 → 研究员撰写具体方案 → 组长审查 → 通过后可进行最终交付（与客户确认、定稿、下发执行）。

适用：化妆品功效测试方案。

以下为组长在审查研究员撰写的具体方案时需逐项关注的点，全部通过后再放行进入交付流程。

一与客户方案设计是否对齐

序号审查点说明
1.1研究目的与客户方案设计中的目的一致，无遗漏或擅自扩大/缩小范围
1.2研究设计类型随机/对照/单多中心等与客户设计一致
1.3产品与使用方法受试化妆品、使用部位/用法/用量与客户设计一致
1.4主要/次要终点客户要求的终点在方案中均有体现，且定义一致
1.5客户特殊要求客户在设计里提出的时间线、访视、问卷、仪器等要求已全部落实进正文

组长判断：具体方案是否完整贯彻了客户的方案设计，有无偏离或遗漏。

二方案内容完整性与逻辑

序号审查点说明
2.1纳入/排除标准表述清晰、可操作；与产品/使用方法匹配；相互无矛盾
2.2基础值入组条件若有，须可量化、可核查，便于执行与数据检查
2.3指标定义每个主要/次要指标有明确定义；评估方法、仪器、问卷与指标一一对应，无悬空指标
2.4访视与评估时点时间轴完整；访视时点与评估时点对应清楚；时间窗口合理、可执行
2.5样本量有说明或计算依据；与“入组人数/完成人数”的预期一致
2.6统计方法主要/次要指标均有对应统计方法；表述无歧义，统计分析可据此执行
2.7测试流程顺序测试流程顺序合理；与访视/评估时点匹配；无逻辑倒置或遗漏环节
2.8测试点位图点位图与方案一致；部位、数量、标注正确，执行方可按图操作

组长判断：方案是否自成一体、逻辑闭环，执行方仅凭本方案即可操作。

三文字与格式

序号审查点说明
3.1前后一致全文术语、缩写、时点、指标名称统一，无前后矛盾
3.2版本与修订若为修订版，修订处有标注或修订说明；版本号、日期正确
3.3可读性语句通顺、层级清晰，客户与执行方易于理解

组长判断：文档是否达到对外/对执行方交付的规范与可读性要求。

四组长审查通过后的交付步骤

审查全部通过后，可按下列顺序进行最终交付：

  1. 与客户确认：将方案发客户审阅，收集修改意见并交由研究员修订，必要时组长复核修订稿。
  2. 定稿：客户确认无异议后，方案定稿，赋予版本号并归档。
  3. 下发执行：将定稿方案下发至执行方/站点，确保其知悉并依此执行。
